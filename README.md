# Tableau Job Market Analysis Dashboard

## Project Overview
This Tableau project analyzes job postings based on salary, experience, qualifications, company size, work type, geographic location, and hiring trends.

## Tasks Completed

### Task 1
Preference vs Work Type (Bar Chart)

### Task 2
Company Size vs Company Name (Scatter Plot)

### Task 3
Top 10 Companies Hiring Data Scientists (Treemap)

### Task 4
Qualification Drilldown Map

### Task 5
India vs Germany Comparison (Stacked Bar Chart)

### Task 6
Work Type Salary Distribution (Box Plot)

## Tools Used
- Tableau
- Microsoft Excel
- GitHub

## Dataset Fields
- Experience
- Qualifications
- Salary Range
- Location
- Country
- Latitude
- Longitude
- Work Type
- Company Size
- Job Posting Date
- Preference
- Job Title
- Role
- Job Portal
- Company Name
